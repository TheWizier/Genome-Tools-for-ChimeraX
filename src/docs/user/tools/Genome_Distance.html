Tool: Genome Distance

This tool is for measuring distances and has three ways of doing so:

"Pairwise Distances" will calculate all pairwise distances between beads within a model.

"Distance Between Models" will calculate all distances between the beads in the two models.

"Distance from Point" will calculate all distances from that point to all beads in the model.

The results are given as an array and can be saved as a .npy file.

A histogram is also generated from the results.

Distance Options

Here one can choose what distance metric to use.

Histogram Options

Here are options for the generated histogram.

"Bins" refer to the amount of bins to be used for the histogram.

If "Use cutoff" is enabled, the the cutoff "min" and "max" values will be used to cut off values outside of that range when generating the histogram.
